# Imosp Cloud App

Imosp Cloud App - React native expo project

# Installation

1. Clone the repo
   ```sh
   git clone https://github.com/saabrinakim/imosp-cloud-app.git
   ```
2. Install yarn packages
   ```sh
   yarn install
   ```
   or
   ```sh
   npm i
   ```
3. Run app
   ```sh
   npx expo start
   ```
